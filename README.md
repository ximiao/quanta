# quanta

# 概述
一个基于lua的分布式游戏服务器引擎框架！

# 优势
- 轻量级
- 简单、易上手
- 稳定性强
- 扩展性强
- 热更新

# 编译
- msvc : 打开项目目录下的sln工程文件，编译即可。
- linux：在项目根目录，执行make all。
